Instrucciones para eclipse:
Versión: Oxygen.3a Release (4.7.3a)

-Abrir Workspace en "C:\"
-Importar el proyecto desde github
-Crear base de datos mysql "babel", usuario root, sin clave
-Ejecutar Script "C:\Babel-Repo\BabelWeb\babel.sql"

  Librerías
  De presentarse errores referentes a las librerías, corregir el buildpath, todos los .jar de "C:\Babel-Repo\BabelWeb\libreriashibernate", "C:\Babel-Repo\BabelWeb" y "C:\Babel-Repo\BabelWeb\WebContent\WEB-INF\lib" son necesarios.
  
  Tomcat
  La versión es 6.0, agregar BabelWeb.
  
  
Otros Frameworks:
Clonar proyecto en "C:\Babel-Repo"
